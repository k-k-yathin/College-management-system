📘 College Management System

A full-stack web application built using Go (Golang), MySQL, and HTML/CSS/JavaScript to manage student, faculty, and course data efficiently.

⸻

🚀 Features
	•	Student registration & management
	•	Faculty management
	•	Course allocation
	•	Database integration with MySQL
	•	RESTful backend using Go

⸻

🛠 Tech Stack
	•	Backend: Go (Golang)
	•	Database: MySQL
	•	Frontend: HTML, CSS, JavaScript
